**Comandos usados**

**Comandos de navegación y sistema**

pwd
Muestra la ruta (ubicación) actual en la terminal.

ls
Lista los archivos y carpetas del directorio actual.

ls -al
Muestra información detallada de archivos y carpetas, incluidos los ocultos.

cd
Cambia de directorio.

mkdir
Crea una nueva carpeta.

touch
Crea un archivo vacío.

vim
Abre un archivo en el editor de texto Vim.

cat nombre_del_archivo
Muestra el contenido de un archivo directamente en la terminal.

^C (Ctrl + C)
Cancela un comando en ejecución.


**Comandos de Git**

git init
Inicializa un repositorio Git en la carpeta actual.

git status
Muestra el estado del repositorio y los cambios pendientes.

git add
Agrega archivos al área de preparación (staging).

git rm --cached
Quita un archivo del área de preparación sin borrarlo del disco.

git commit -m
Guarda los cambios preparados en el historial del repositorio.

git branch -M
Cambia el nombre de la rama principal.

git log
Muestra el historial de commits del repositorio.

git config --global user.name
Configura el nombre del autor de los commits.

git config --global user.email
Configura el correo del autor de los commits.

git config --list
Muestra todas las configuraciones activas de Git.

git config --global init.defaultbranch
Define el nombre por defecto de la rama inicial.
